**Code Review AI**

AI-powered code review platform that analyzes your code and provides suggestions for improvements, best practices, efficiency, and security.

Tech Stack
Frontend:
React, Vite, Prism.js, react-simple-code-editor, react-markdown

Backend:
Node.js, Express.js, Google Generative AI API

Other:
Axios, CORS, dotenv

Features
Write or paste code in the editor

Get AI-generated code review with suggestions and recommended fixes

Highlight issues, inefficiencies, and improvements in the code
